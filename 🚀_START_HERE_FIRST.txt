## 📞 Troubleshooting

### During Push
- **"Git not found"** → Install from https://git-scm.com/download/win
- **"Permission denied"** → Use GitHub token, not password
- **"Repository not found"** → Check GitHub URL and visibility

### During Deployment
- **"Build failed"** → Check env vars in Vercel
- **"App blank"** → Check browser console for errors
- **"Service Worker issue"** → Clear cache, restart browser

**See detailed guides:** GIT_SETUP_GUIDE.md, GITHUB_DEPLOYMENT_GUIDE.md
